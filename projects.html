Calculadora de IMC

Neste projeto, foi criada uma página web interativa para calcular o IMC do usuário. Basta preencher os dados de peso e altura, e o sistema gera o resultado automaticamente. O desenvolvimento utilizou HTML, CSS e JavaScript para a estrutura, estilo e lógica de cálculo.
